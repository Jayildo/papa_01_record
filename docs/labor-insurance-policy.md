# 노무비 보험 판정 정책

기준일: 2026-04-08

현재 앱 정책은 `법령상 자동 판정 가능한 범위`와 `사용자 입력이 필요한 범위`를 구분한다.

## 자동 계산

- 일용 소득세: `max(0, 총 지급액 - 150,000) x 6% x 45%`
- 지방소득세: `소득세 x 10%`
- 고용보험: `총 지급액 x 0.9%`
- 국민연금: `천원 미만 절사한 월 보수 x 4.75%`
- 건강보험: `총 지급액 x 3.595%`
- 장기요양보험: `건강보험료 x 13.14%`

## 자동 판정 조건

- 국민연금
  - `고용기간 = 1개월 이상`
  - 그리고 `월 근로일수 8일 이상`
  - 또는 `일반사업장 + 월 근로시간 60시간 이상`
- 건강보험/장기요양보험
  - `고용기간 = 1개월 이상`

## 수동 입력 유지 항목

- 국민연금, 건강보험, 장기요양보험, 기타공제는 모두 수동 입력값을 추가 반영한다.
- 자동 판정이 맞지 않는 예외 사례는 수동 입력으로 보정한다.

## 주의

- 국민연금은 사업장 유형과 근로시간, 예외 사유에 따라 실제 적용 여부가 달라질 수 있다.
- 건강보험도 실제 자격 취득일 기준 확인이 필요한 경우가 있다.
- 앱은 현재 `월 단위 노무비 계산`에 필요한 실무용 보수적 자동화까지만 구현했다.
